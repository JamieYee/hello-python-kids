## 四、核心知识讲解

### 1）什么是循环

循环是一种**重复执行一段代码**的机制。

```python
# ❌ 不用循环（太麻烦）
print(1)
print(2)
print(3)
# ... 一直到
print(100)

# ✅ 用循环（优雅！）
for i in range(1, 101):
    print(i)
```

### 2）For 循环（计数循环）

#### 2.1 基础 for 循环

语法：循环一个**已知的集合**（如列表、字符串、范围）

```python
# 循环列表
fruits = ["苹果", "香蕉", "橙子"]

for fruit in fruits:
    print(f"我喜欢吃{fruit}")

# 输出：
# 我喜欢吃苹果
# 我喜欢吃香蕉
# 我喜欢吃橙子
```

#### 2.2 使用 range() 函数

`range(start, end, step)` 生成一个数字序列。

```python
# range(5)：0, 1, 2, 3, 4
for i in range(5):
    print(i)
# 输出: 0 1 2 3 4

# range(1, 11)：1, 2, ..., 10
for i in range(1, 11):
    print(i)

# range(0, 10, 2)：0, 2, 4, 6, 8
for i in range(0, 10, 2):
    print(i)

# range(10, 0, -1)：10, 9, 8, ..., 1
for i in range(10, 0, -1):
    print(i)
```

#### 2.3 循环与变量名

```python
# 变量名可以是任何单词，不一定是 i
scores = [90, 85, 92, 78]

# for score in scores:
#     print(score)

# ✅ 意思一样，变量名更清晰
for current_score in scores:
    print(current_score)
```

#### 2.4 enumerate()：获得索引和值

```python
students = ["小明", "小红", "小刚"]

# 方式1：只要值
for student in students:
    print(student)

# 方式2：需要索引和值
for index, student in enumerate(students):
    print(f"{index + 1}. {student}")

# 输出：
# 1. 小明
# 2. 小红
# 3. 小刚

# 指定起始索引
for index, student in enumerate(students, start=1):
    print(f"{index}. {student}")
```

#### 2.5 zip()：同时循环多个列表

```python
names = ["Alice", "Bob", "Charlie"]
ages = [20, 21, 22]
cities = ["北京", "上海", "深圳"]

for name, age, city in zip(names, ages, cities):
    print(f"{name} 今年 {age} 岁，住在{city}")

# 输出：
# Alice 今年20岁，住在北京
# Bob 今年21岁，住在上海
# Charlie 今年22岁，住在深圳
```

### 3）While 循环（条件循环）

#### 3.1 基础 while 循环

语法：只要**条件为真**，就一直循环。

```python
count = 0

while count < 5:
    print(f"计数: {count}")
    count = count + 1  # ⚠️ 必须每次改变条件！

# 输出：
# 计数: 0
# 计数: 1
# 计数: 2
# 计数: 3
# 计数: 4
```

#### 3.2 While vs For

| 用途 | 推荐用 |
| --- | --- |
| 循环一个已知的序列 | `for` |
| 循环次数未知 | `while` |
| 循环到满足某条件 | `while` |

```python
# ✅ 用 for 循环 1-10
for i in range(1, 11):
    print(i)

# ❌ 用 while 循环 1-10（可以但多余）
i = 1
while i <= 10:
    print(i)
    i += 1

# ✅ 用 while 循环（猜数字游戏）
secret = 7
guess = None

while guess != secret:
    guess = int(input("猜一个数字 (1-10): "))
    if guess < secret:
        print("太小了")
    elif guess > secret:
        print("太大了")
    else:
        print("猜对了！")
```

### 4）循环控制：break 和 continue

#### 4.1 break：立即退出循环

```python
# 分场景：找到某个元素就停止
items = [1, 2, 3, 4, 5, 6]
target = 4

for item in items:
    if item == target:
        print(f"找到了 {target}")
        break  # 停止循环
    print(item)

# 输出：
# 1
# 2
# 3
# 找到了 4
```

#### 4.2 continue：跳过本次迭代

```python
# 场景：打印所有奇数
for i in range(1, 11):
    if i % 2 == 0:
        continue  # 跳过偶数
    print(i)

# 输出：
# 1
# 3
# 5
# 7
# 9
```

#### 4.3 break vs continue

```python
print("=== break ===")
# break: 完全退出循环
for i in range(1, 6):
    if i == 3:
        break
    print(i)
# 输出: 1 2

print("\n=== continue ===")
# continue: 跳过当前，继续下一次
for i in range(1, 6):
    if i == 3:
        continue
    print(i)
# 输出: 1 2 4 5
```

### 5）嵌套循环

#### 5.1 循环中的循环

```python
# 打印一个 3x3 的矩阵
for i in range(1, 4):
    for j in range(1, 4):
        print(f"({i}, {j})", end=" ")
    print()  # 换行

# 输出：
# (1, 1) (1, 2) (1, 3)
# (2, 1) (2, 2) (2, 3)
# (3, 1) (3, 2) (3, 3)

# 99 乘法表
for i in range(1, 10):
    for j in range(1, 10):
        print(f"{i}×{j}={i*j}", end="\t")
    print()
```

#### 5.2 嵌套循环中的 break

```python
# break 只会退出最内层循环
for i in range(1, 4):
    print(f"第 {i} 行：", end="")
    for j in range(1, 4):
        if j == 2:
            break
        print(j, end=" ")
    print()

# 输出：
# 第 1 行：1
# 第 2 行：1
# 第 3 行：1
```

### 6）常见错误与陷阱

#### 6.1 死循环（无限循环）

```python
# ❌ 死循环！条件永远为真
# while True:
#     print("永远不会停止")

# ✅ 修正：添加退出条件
count = 0
while True:
    print(count)
    count += 1
    if count >= 3:
        break
```

#### 6.2 遍历时修改列表

```python
# ❌ 危险！遍历时删除元素
numbers = [1, 2, 3, 4, 5]
# for num in numbers:
#     if num == 3:
#         numbers.remove(num)  # 会跳过元素！

# ✅ 方式1：创建副本
numbers = [1, 2, 3, 4, 5]
for num in numbers[:]:  # 遍历副本
    if num == 3:
        numbers.remove(num)
print(numbers)  # [1, 2, 4, 5]

# ✅ 方式2：用列表推导式
numbers = [1, 2, 3, 4, 5]
numbers = [n for n in numbers if n != 3]
print(numbers)  # [1, 2, 4, 5]
```

#### 6.3 遍历与索引混淆

```python
# ❌ 遍历时不应该手动改变索引
fruits = ["苹果", "香蕉", "橙子"]
# for fruit in fruits:
#     i += 1  # ❌ i 没定义！

# ✅ 用 enumerate()
for i, fruit in enumerate(fruits):
    print(f"{i}: {fruit}")
```

### 任务1：打印图案

```python
# 任务1.1：打印一个三角形
print("=== 三角形 ===")
for i in range(1, 6):
    print("*" * i)

# 输出：
# *
# **
# ***
# ****
# *****

# 任务1.2：打印一个倒三角形
print("\n=== 倒三角形 ===")
for i in range(5, 0, -1):
    print("*" * i)

# 任务1.3：打印一个菱形
print("\n=== 菱形 ===")
n = 4
for i in range(1, n + 1):
    print(" " * (n - i) + "*" * (2 * i - 1))
for i in range(n - 1, 0, -1):
    print(" " * (n - i) + "*" * (2 * i - 1))
```

### 任务2：数字游戏

```python
# 任务2.1：求和
print("=== 求和 ===")
total = 0
for i in range(1, 101):
    total += i
print(f"1 到 100 的和: {total}")  # 5050

# 任务2.2：阶乘
print("\n=== 阶乘 ===")
n = 5
factorial = 1
for i in range(1, n + 1):
    factorial *= i
print(f"{n}! = {factorial}")  # 120

# 任务2.3：找出质数
print("\n=== 找质数 ===")
def is_prime(num):
    if num < 2:
        return False
    for i in range(2, int(num ** 0.5) + 1):
        if num % i == 0:
            return False
    return True

primes = [n for n in range(1, 31) if is_prime(n)]
print(f"1-30 中的质数: {primes}")
```

### 任务3：交互循环

```python
# 任务：简单计算器（输入 'q' 退出）
print("=== 简单计算器 ===")

while True:
    num1 = input("输入第一个数字 (或 'q' 退出): ")
    
    if num1.lower() == 'q':
        print("再见！")
        break
    
    operator = input("输入操作符 (+, -, *, /): ")
    num2 = input("输入第二个数字: ")
    
    try:
        num1, num2 = float(num1), float(num2)
        
        if operator == "+":
            result = num1 + num2
        elif operator == "-":
            result = num1 - num2
        elif operator == "*":
            result = num1 * num2
        elif operator == "/":
            if num2 == 0:
                print("❌ 不能除以零")
                continue
            result = num1 / num2
        else:
            print("❌ 无效的操作符")
            continue
        
        print(f"结果: {result}\n")
    except ValueError:
        print("❌ 请输入有效的数字\n")
```

### 任务4：列表处理

```python
# 任务：处理成绩列表
scores = [92, 85, 78, 95, 88, 76, 92, 91]

print(f"所有分数: {scores}")

# 计算平均分
average = sum(scores) / len(scores)
print(f"平均分: {average:.2f}")

# 找出及格人数（>=60）
passed = 0
for score in scores:
    if score >= 60:
        passed += 1
print(f"及格人数: {passed}")

# 找出不及格的分数
print("不及格分数: ", end="")
for score in scores:
    if score < 60:
        print(score, end=" ")
print()

# 分数等级统计
grades = {"A": 0, "B": 0, "C": 0, "D": 0, "F": 0}
for score in scores:
    if score >= 90:
        grades["A"] += 1
    elif score >= 80:
        grades["B"] += 1
    elif score >= 70:
        grades["C"] += 1
    elif score >= 60:
        grades["D"] += 1
    else:
        grades["F"] += 1

print(f"等级分布: {grades}")
```

### C 级（挑战）

**实现一个"FizzBuzz"游戏：**

```python
"""
规则：
- 3 的倍数打印 "Fizz"
- 5 的倍数打印 "Buzz"
- 既是 3 也是 5 的倍数打印 "FizzBuzz"
- 其他打印数字本身
"""

for i in range(1, 101):
    output = ""
    
    if i % 3 == 0:
        output += "Fizz"
    if i % 5 == 0:
        output += "Buzz"
    if not output:
        output = str(i)
    
    print(output)

# 输出（前20行）：
# 1 2 Fizz 4 Buzz Fizz 7 8 Fizz Buzz
# 11 Fizz 13 14 FizzBuzz 16 17 Fizz 19 Buzz
```
