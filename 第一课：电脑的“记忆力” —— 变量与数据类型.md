## 四、核心知识讲解

### 1）什么是变量（Variables）

- **变量名**：盒子标签（如 `score`）
- **变量值**：盒子里的内容（如 `100`）
- **赋值符号**：`=` 表示“把右边放进左边”

示例：

```python
score = 100
score = 120  # 变量可以被更新
```

课堂提问：

- `name = "小明"` 中，变量名是什么？
- `level = 3` 中，变量值是什么？

### 2）基础数据类型（Data Types）

| 类型名称 | 英文名 | 缩写 | 生活类比 | 例子 |
| --- | --- | --- | --- | --- |
| 整数 | Integer | `int` | 数人数、数苹果 | `10`, `-5`, `0` |
| 浮点数 | Floating-point | `float` | 测量身高、体重 | `1.75`, `9.99` |
| 布尔值 | Boolean | `bool` | 灯的开关（开 / 关） | `True`, `False` |
| 字符串 | String | `str` | 名字、一段话 | `"Hello"`, `'你好'` |

快问快答（全班）：

- `66` 是什么类型？
- `66.0` 是什么类型？
- `"66"` 是什么类型？

### 3）深入字符串（String）

#### 3.1 字符串定义

```python
name = "小明"          # 双引号
motto = 'Keep Coding'  # 单引号
story = """从前有座山，
山里有座庙..."""        # 三引号（多行文本）
```

#### 3.2 字符串操作

- **拼接（`+`）**：`"Super" + "man"` ➔ `"Superman"`
- **重复（`*`）**：`"哈" * 3` ➔ `"哈哈哈"`
- **长度（`len`）**：`len("Python")` ➔ `6`

#### 3.3 索引（Index）

规则：索引从 `0` 开始。

```python
word = "PYTHON"
# 索引：P(0) Y(1) T(2) H(3) O(4) N(5)

print(word[0])  # P
print(word[3])  # H
```

易错提醒：

- 第 1 个字符不是索引 1，而是索引 0。

### 4）类型转换（Type Conversion）

```python
age = 12
# print("我今年" + age + "岁")  # ❌ 类型不一致

print("我今年" + str(age) + "岁")  # ✅ 转成字符串
```

讲解重点：

- 字符串和数字不能直接用 `+` 拼接。
- 用 `str()` 可以把数字“变身”为字符串。

## 五、课堂练习：我的英雄卡片

请学生先运行，再按要求改值。

```python
# 1. 定义不同类型变量
hero_name = "闪电麦昆"      # str
hero_level = 99            # int
hero_speed = 325.5         # float
is_ready = True            # bool

# 2. 打印英雄信息
print("-" * 20)
print(f"英雄角色: {hero_name}")
print(f"当前等级: {hero_level}")
print(f"最高时速: {hero_speed} km/h")
print(f"准备就绪: {is_ready}")
print("-" * 20)

# 3. 尝试计算
next_level = hero_level + 1
print(f"升级后等级将变为: {next_level}")
```

任务要求：

1. 把角色名改成你喜欢的角色。
2. 把等级改成任意整数。
3. 把速度改成任意小数。
4. 把准备状态改成 `False` 看输出变化。
