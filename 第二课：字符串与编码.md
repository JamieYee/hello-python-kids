## 四、核心知识讲解

### 1）什么是编码（Encoding）

#### 1.1 编码的本质

电脑只认识 `0` 和 `1`，它不认识"A"、"中"这样的字符。
所以我们需要一个"翻译表"，把每个字符转换成数字。这个"翻译表"就是**编码**。

**过程：**

```
字符 "A" → 编码 → 数字 65 → 二进制 01000001 → 存储在硬盘
```

#### 1.2 编码系统家族

我们需要记住两种编码：

| 编码名 | 能表示什么 | 什么时候用 |
| --- | --- | --- |
| ASCII | 只有英文、数字、符号 | 很少用了 |
| UTF-8 | 全世界所有文字（推荐！） | 现在都用这个 |

### 2）ASCII 编码故事（1960年）

ASCII = American Standard Code for Information Interchange

- **A** → 65
- **B** → 66
- **a** → 97
- **0** → 48
- **空格** → 32

```python
# 查看 ASCII 值
ord("A")   # ➔ 65
chr(65)    # ➔ "A"
```

快问快答：

- `ord("我")` 输出什么？（不在 ASCII 范围内）
- `chr(90)` 输出什么？

### 3）现代编码：UTF-8

**UTF-8 是什么？** 它是一个超级聪明的编码方式，可以表示世界上所有的文字——英文、中文、日文、阿拉伯文，甚至 emoji 都可以！

**UTF-8 的秘密：** 不同的字符用不同的字节数

- 英文字母 A → 1 个字节
- 中文字符 "中" → 3 个字节  
- Emoji 😀 → 4 个字节

```python
"Hello".encode("utf-8")  
# b'Hello'  （英文，简单）

"你好".encode("utf-8")  
# b'\xe4\xbd\xa0\xe5\xa5\xbd'  （中文，3字节每个）
```

**简单的理解方式：** 就像邮件一样，短信用一张邮票，长信用多张邮票。UTF-8 就是这样根据需要调整。

### 4）Python 中的编码操作

#### 4.1 字符串的 `encode()` 方法

把**字符串**转成**字节**

**语法：**

```python
string.encode(编码名称)
```

**示例：**

```python
text = "Hello World"
bytes_en = text.encode("utf-8")
print(bytes_en)  # b'Hello World'
print(type(bytes_en))  # <class 'bytes'>

text_cn = "你好"
bytes_cn = text_cn.encode("utf-8")
print(bytes_cn)  # b'\xe4\xbd\xa0\xe5\xa5\xbd'

# 试试用 GBK 编码
bytes_gbk = text_cn.encode("gbk")
print(bytes_gbk)  # b'\xc4\xe3\xba\xc3'  （不同的字节）
```

#### 4.2 字节的 `decode()` 方法

把**字节**转回**字符串**

**语法：**

```python
bytes_object.decode(编码名称)
```

**示例：**

```python
# 编码
message = "Python 编程"
encoded = message.encode("utf-8")
print(encoded)  # b'Python \xe7\xbc\x96\xe7\xa8\x8b'

# 解码
decoded = encoded.decode("utf-8")
print(decoded)  # Python 编程
print(decoded == message)  # True
```

### 5）编码错误与修复

#### 5.1 常见错误1：编码与解码不匹配

```python
# ❌ 错误示范
text = "你好"
encoded = text.encode("utf-8")  # 用 UTF-8 编码

# 错误地用 GBK 解码
# decoded = encoded.decode("gbk")  # UnicodeDecodeError!

# ✅ 正确做法
decoded = encoded.decode("utf-8")
print(decoded)  # 你好
```

**规则：用什么编码，就用什么解码。**

#### 5.2 常见错误2：字符串包含无法编码的字符

```python
# 某些编码无法表示所有字符
emoji = "你好😀"

# ASCII 无法编码中文和 emoji
# emoji.encode("ascii")  # UnicodeEncodeError!

# UTF-8 可以编码所有字符
print(emoji.encode("utf-8"))  # b'\xe4\xbd\xa0\xe5\xa5\xbd\xf0\x9f\x98\x80'
```

#### 5.3 修复的办法

**最简单的办法：用 UTF-8 就不会出错！**

因为 UTF-8 可以表示所有字符，所以只要始终用 UTF-8 编码，就不会有问题：

```python
problem_text = "Hello 你好 😀"

# ✅ 用 UTF-8 编码，一成不变！
result = problem_text.encode("utf-8")
print(result)  # b'Hello \xe4\xbd\xa0\xe5\xa5\xbd \xf0\x9f\x98\x80'

# 解码回来
decoded = result.decode("utf-8")
print(decoded)  # Hello 你好 😀
```

**记住这个金律：** 编码和解码都用 UTF-8，就永远不会出错！

### 6）实用技巧：查看字符的字节数

```python
def byte_counter(char):
    """查看一个字符有多少个字节"""
    byte_data = char.encode('utf-8')
    print(f"字符: {char}")
    print(f"UTF-8 字节: {byte_data}")
    print(f"字节数: {len(byte_data)}")
    print()

byte_counter("A")   # 英文，1个字节
byte_counter("中")  # 中文，3个字节
byte_counter("😀")  # Emoji，4个字节
```

### 任务2：计算不同文本的字节数

```python
def count_bytes(text):
    """计算文本有多少个字节"""
    bytes_data = text.encode("utf-8")
    print(f"文本: {text}")
    print(f"字符数: {len(text)}")
    print(f"字节数: {len(bytes_data)}")
    print()

# 测试不同的文本
count_bytes("Hello")      # 英文：字符数=字节数
count_bytes("你好")       # 中文：字符数 < 字节数
count_bytes("Hello你好")  # 混合：看看有多少字节
count_bytes("😀")        # Emoji：4个字节
```

### C 级（挑战）

自己设计一个简单的密码游戏：

```python
def simple_secret(text):
    """
    简单密码：显示每个字符的字节值（用逗号分开）
    """
    bytes_list = text.encode("utf-8")
    return ",".join(str(b) for b in bytes_list)

def reveal_secret(secret_code):
    """
    破译密码：把字节值转回文字
    """
    bytes_list = [int(b) for b in secret_code.split(",")]
    bytes_data = bytes(bytes_list)
    return bytes_data.decode("utf-8")

# 游戏开始
message = "你好"
encrypted = simple_secret(message)
print(f"原文: {message}")
print(f"密码: {encrypted}")

decrypted = reveal_secret(encrypted)
print(f"破译: {decrypted}")
```
