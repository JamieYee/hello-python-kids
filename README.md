# 🚀 Hello Python Kids!

欢迎来到 Hello Python Kids！这是一套为小学和初中学生设计的 Python 编程教学材料，注重从生活实例出发，让学生轻松理解编程概念。

---

## 🎯 课程目标

学完这 6 节课，你将能够：

| 目标 | 描述 |
| --- | --- |
| 🧠 **培养编程思维** | 学会像程序员一样分析问题、拆解问题、解决问题 |
| 💻 **掌握 Python 基础** | 熟练运用变量、数据类型、条件、循环等核心语法 |
| 🎮 **实现游戏化项目** | 通过实战练习，独立编写有趣的小程序（英雄卡片、背包系统等） |
| 📊 **理解数据结构** | 理解和使用 List、Tuple、Dict、Set 等常用数据结构 |
| 🚀 **迈向高级编程** | 为学习函数、模块、文件操作等高级特性打下坚实基础 |

## 📚 课程大纲 (6 节课)

本项目分为 6 个核心模块，建议按顺序学习。每课为**40～45 分钟**，包含讲解、代码演示、实战练习和课后挑战。

| 课号 | 主题 | 核心内容 | 学习重点 |
| :---: | --- | --- | :---: |
| **L1** | [📝 电脑的"记忆力"](第一课：变量与数据类型.md) | 变量、int、float、bool、str | 理解数据类型和变量赋值 |
| **L2** | [💬 让电脑"开口说话"](第二课：字符串与编码.md) | 字符串操作、编码、解码 | 掌握字符串处理和字符编码原理 | 
| **L3** | [📦 数据的"收纳盒"](第三课：List%20与%20Tuple.md) | List、Tuple、索引、切片 | 学会使用集合数据结构 |
| **L4** | [🧠 大脑的"决策树"](第四课：条件判断与模式匹配.md) | if/elif/else、逻辑运算、match | 掌握条件判断和模式匹配 | 
| **L5** | [🔄 枯燥工作的"终结者"](第五课：循环.md) | for、while、break、continue | 理解循环控制和流程优化 |
| **L6** | [🔍 超级"查找字典"](第六课：Dict%20与%20Set.md) | Dict、Set、键值对、去重 | 学会高效地组织和查找数据 |
