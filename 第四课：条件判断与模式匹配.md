## 四、核心知识讲解

### 1）布尔值（Boolean）与比较运算符

#### 1.1 什么是布尔值

布尔值只有两个值：`True` 和 `False`（真与假）。

```python
# 布尔值
is_online = True
is_admin = False

print(type(True))   # <class 'bool'>
print(type(False))  # <class 'bool'>
```

#### 1.2 六大比较运算符

| 运算符 | 含义 | 例子 | 结果 |
| --- | --- | --- | --- |
| `==` | 相等 | `5 == 5` | `True` |
| `!=` | 不相等 | `5 != 3` | `True` |
| `>` | 大于 | `5 > 3` | `True` |
| `<` | 小于 | `2 < 5` | `True` |
| `>=` | 大于等于 | `5 >= 5` | `True` |
| `<=` | 小于等于 | `3 <= 5` | `True` |

```python
# 注意：比较结果是布尔值！
print(10 > 5)      # True
print(10 == 20)    # False
print("abc" == "abc")  # True
print([1, 2] != [1, 2, 3])  # True

# 常见错误
age = 18
# if age = 18:   # ❌ 这是赋值，不是比较！
if age == 18:   # ✅ 这是比较
    print("成年了")
```

快问快答：

- `5 > 3 and 3 < 2` 是 True 还是 False？
- `len("python") == 6` 是 True 还是 False？

### 2）条件语句：if、elif、else

#### 2.1 基础 if 语句

```python
score = 85

if score >= 90:
    print("优秀！")

# 如果条件不满足，什么都不做
```

#### 2.2 if-else 语句

```python
age = 15

if age >= 18:
    print("成年人")
else:
    print("未成年人")
```

#### 2.3 if-elif-else 语句（多分支）

```python
score = 75

if score >= 90:
    grade = "A"
elif score >= 80:
    grade = "B"
elif score >= 70:
    grade = "C"
elif score >= 60:
    grade = "D"
else:
    grade = "F"

print(f"等级: {grade}")  # 等级: C
```

**重要规则：**

- 多个 `elif` 按顺序检查，一旦满足就停止。
- `else` 总是最后的 fallback。

#### 2.4 嵌套条件语句

```python
age = 20
has_license = True

if age >= 18:
    if has_license:
        print("可以开车")
    else:
        print("年纪够但没驾照")
else:
    print("太年轻了")
```

### 3）逻辑运算符：and、or、not

#### 3.1 AND（与）- 全部为真才为真

```python
age = 18
has_money = True
has_ticket = True

if age >= 18 and has_money and has_ticket:
    print("可以看电影")  # ✅ 输出
else:
    print("不能看")

# 任何一个条件为 False，整体就是 False
age = 17
if age >= 18 and has_money:
    print("这行不会执行")  # ❌
```

**真值表（AND）：**

| A | B | A and B |
| --- | --- | --- |
| T | T | T |
| T | F | F |
| F | T | F |
| F | F | F |

#### 3.2 OR（或）- 至少一个为真就为真

```python
can_drive = False
can_bike = True

if can_drive or can_bike:
    print("可以出行")  # ✅ 输出

# 只要一个条件为 True，整体就是 True
season = "冬天"
is_weekend = True

if season == "夏天" or is_weekend:
    print("可以出去玩")  # ✅ 输出
```

**真值表（OR）：**

| A | B | A or B |
| --- | --- | --- |
| T | T | T |
| T | F | T |
| F | T | T |
| F | F | F |

#### 3.3 NOT（非）- 反转真值

```python
is_raining = False

if not is_raining:
    print("天气很好")  # ✅ 输出

# 等价写法
if is_raining == False:
    print("天气很好")  # ✅ 也输出
```

#### 3.4 组合复杂条件

```python
# 你有资格参加比赛的条件：
# - 年龄 15-25 岁
# - 有参赛费 或 学校代表队
# - 没有禁赛

age = 18
has_fee = False
is_representative = True
is_banned = False

if (age >= 15 and age <= 25) and (has_fee or is_representative) and not is_banned:
    print("符合参赛条件")  # ✅ 输出
```

### 4）模式匹配（Pattern Matching，Python 3.10+）

#### 4.1 什么是模式匹配

`match` 语句让你根据值的**结构**来分支，比 if-elif-else 更优雅。

```python
# 旧方式（if-elif）
response = "yes"

if response == "yes":
    print("同意")
elif response == "no":
    print("拒绝")
elif response == "maybe":
    print("不确定")
else:
    print("无效回复")

# ✨ 新方式（match）
match response:
    case "yes":
        print("同意")
    case "no":
        print("拒绝")
    case "maybe":
        print("不确定")
    case _:  # _ 是默认情况（类似 else）
        print("无效回复")
```

#### 4.2 模式匹配的优势

```python
# 1. 匹配多个值
command = "help"

match command:
    case "start" | "run" | "go":  # | 表示"或"
        print("启动程序")
    case "stop" | "quit" | "exit":
        print("停止程序")
    case _:
        print("未知命令")
```

#### 4.3 匹配数据结构

```python
# 匹配列表
point = [3, 3]

match point:
    case [0, 0]:
        print("原点")
    case [x, 0]:
        print(f"x 轴上，x={x}")
    case [0, y]:
        print(f"y 轴上，y={y}")
    case [x, y]:
        print(f"点({x}, {y})")

# 交互：point = [3, 3] 时输出：点(3, 3)

# 匹配元组
rgb = (255, 128, 0)

match rgb:
    case (255, 0, 0):
        print("红色")
    case (0, 255, 0):
        print("绿色")
    case (0, 0, 255):
        print("蓝色")
    case (r, g, b):
        print(f"RGB({r}, {g}, {b})")
```

#### 4.4 带条件的模式匹配（Guard）

```python
age = 25

match age:
    case age if age < 13:
        print("儿童")
    case age if age < 18:
        print("未成年")
    case age if age < 65:
        print("成年人")
    case _:
        print("老年人")
```

### 5）常见模式与陷阱

#### 5.1 条件的短路求值

```python
# AND 中，一旦左边是 False，右边就不计算
print(False and print("hello"))  # 不输出 hello（短路）
print(True and print("hello"))   # 输出 hello

# OR 中，一旦左边是 True，右边就不计算
print(True or print("world"))   # 不输出 world（短路）
print(False or print("world"))  # 输出 world
```

#### 5.2 真值性（Truthiness）

在 Python 中，某些值被认为是"真"或"假"：

```python
# 假值（Falsy）
if 0:       # False
    print("0 是假")

if "":      # False
    print("空字符串是假")

if []:      # False
    print("空列表是假")

if None:    # False
    print("None 是假")

# 真值（Truthy）
if 1:       # True
    print("1 是真")

if "hello": # True
    print("非空字符串是真")

if [1, 2]: # True
    print("非空列表是真")
```

**常见的假值：** `0`、`""`、`[]`、`()`、`None`、`False`

def get_upgrade_reward(level):
    """根据升级后的等级获得对应奖励"""
    match level:

### 任务4：综合练习：宝可梦属性克制

```python
def battle_result(my_type, enemy_type):
    """判定属性克制关系"""
    
    match (my_type, enemy_type):
        # 我方获胜的情况
        case ("水", "火") | ("火", "草") | ("草", "水"):
            return "克制！我方获胜"
        
        # 我方劣势的情况
        case ("火", "水") | ("草", "火") | ("水", "草"):
            return "被克制，我方劣势"
        
        # 相同属性
        case (a, b) if a == b:
            return "属性相同，50-50"
        
        # 其他
        case _:
            return "属性中立"

# 测试
battles = [
    ("水", "火"),
    ("火", "水"),
    ("火", "火"),
    ("雷", "水")
]

for my, enemy in battles:
    print(f"我({my}) vs 对手({enemy)}: {battle_result(my, enemy)}")
```

### C 级（挑战）

**实现一个"石头剪刀布"游戏的胜负判定：**

```python
def rps_winner(player1, player2):
    """
    返回赢家：'玩家1' / '玩家2' / '平局'
    """
    
    # 使用 match 实现
    match (player1, player2):
        case ("石头", "剪刀") | ("剪刀", "布") | ("布", "石头"):
            return "玩家1 赢"
        case (a, b) if a == b:
            return "平局"
        case _:
            return "玩家2 赢"

# 测试
test_cases = [
    ("石头", "剪刀"),
    ("布", "石头"),
    ("剪刀", "剪刀"),
    ("剪刀", "石头")
]

for p1, p2 in test_cases:
    result = rps_winner(p1, p2)
    print(f"{p1} vs {p2}: {result}")
```
